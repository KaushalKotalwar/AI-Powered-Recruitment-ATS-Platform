# AI-Powered-Recruitment-ATS-Platform
An intelligent Applicant Tracking System that uses AI to parse resumes, match skills, and rank candidates efficiently.
📌 Features

✅ Resume Parsing (PDF/DOC)
✅ AI-Based Candidate Ranking
✅ Skill Matching with Job Description
✅ Hiring Pipeline Dashboard
✅ Interview Scheduling

🛠 Tech Stack
Frontend: React.js
Backend: FastAPI / Node.js
Database: PostgreSQL
AI/NLP: Python (spaCy / NLTK)
📸 Screenshots





⚙️ Installation
git clone https://github.com/your-username/ai-recruitment-ats.git
cd ai-recruitment-ats
Install Dependencies
cd client
npm install

cd ../server
npm install
Run Project
# Start backend
cd server
npm start

# Start frontend
cd client
npm start
📊 How It Works
Upload resume
System extracts skills & data
Matches with job description
Ranks candidates based on score
